STACKED CMOS TCAD — 软件说明书

版本 0.5 · 作者:Yu Rui · 开源(Apache-2.0)

面向 CFET 与堆叠环栅(GAA)纳米片晶体管的开源器件级 TCAD 系统, 基于 Python + DEVSIM(漂移-扩散求解器)+ gmsh(网格),桌面工作台 对标 Synopsys Sentaurus 工具链。本说明书概述软件做什么、适用于何处、 与主流工具的对标结果,以及能力边界。具体操作步骤见 Help → User Guide(用户指南)。

1. 软件简介

STACKED CMOS TCAD 在 2D/3D 网格上求解半导体漂移-扩散方程,由一份 简短的 YAML 描述或导入的 CAD/网格文件出发,预测先进 CMOS 器件的电学 特性 — 纳米片 GAA FET、互补 FET(CFET)、垂直堆叠器件。定位 3 nm 及以下节点,面向科研、方法学开发与教学。

2. 软件特性

器件结构

  • nanosheet_2d — 2D 双栅沿沟道剖面(标准的快速 2D GAA 近似)
  • gaa_3d — 完整 3D 单纳米片环栅
  • cfet_2d / cfet_3d — 互补 FET 堆叠 (nFET 与 pFET 上下叠置)
  • 多沟道复制 — 每器件多个并排 fin(n_fins) 或多个叠层 sheet(n_stacked_sheets)
  • external — 导入用户自备的 gmsh MSH 2.2 网格, 或在软件内转换 STEP(.step)CAD 装配体

物理模型

  • 漂移-扩散,Scharfetter–Gummel 离散
  • 迁移率:Caughey–Thomas(掺杂)、CVT/Lombardi 垂直场退化、 高场速度饱和
  • 密度梯度量子修正(限域引起的 Vt 偏移、element 级电流)
  • SiGe(x) 组分插值;按晶面取向的迁移率标定 (mobility_scale_n/p)

实验与参数提取

  • 转移特性(Id–Vg)、输出特性(Id–Vd)、CFET 共栅转移, 以及反相器电压传输特性(器件/电路混合牛顿求解)
  • 品质因数:Vt(恒流法与最大 gm 法)、亚阈摆幅、 DIBL、Ion/Ioff、反相器增益与噪声容限

工作流与平台

  • Sentaurus Workbench 风格的实验表,每行自带 Run/Stop/Sweep/Structure、实时进度百分比、并行扫描 / DOE、 CSV 设计点导入
  • 交互式 3D VTK 渲染;几何导入/导出(STEP 入;STL/PLY/VTP/OBJ 出)
  • 跨平台:Linux 与 Windows 独立可执行程序 (Windows 用 MKL PARDISO,Linux 用 OpenBLAS+UMFPACK)
  • Apache-2.0 开源;可脚本化的 Python API 与命令行

3. 适用范围

适用于:

  • 先进 CMOS(GAA 纳米片、CFET、堆叠器件,3 nm 及以下)的 器件级 TCAD
  • 方法学开发、DOE 参数扫描、架构权衡研究(如 fin 型 vs sheet 型 CFET)
  • 以透明开源的代码库教学漂移-扩散输运与量子修正

不适用于:

  • 量产工艺级或版图级签核
  • 与厂商专有格式互操作(TDR/BND/CMD、GDSII/OASIS)
  • 应力工程或环形振荡器电路级签核

4.2 器件级复现结果

fin 型(FBC)vs sheet 型(SBC)CFET 的对比,对标 Jiang 等 (Applied Materials)论文 《Complementary FET Device and Circuit Level Evaluation Using Fin-Based and Sheet-Based Configurations Targeting 3nm Node and Beyond》,其数据来自商用(Sentaurus 级)TCAD 流程。以 Ioff=1 nA 的恒流 Ion(与论文口径一致), 四项电流增量方向全部与论文一致:

构型ΔIon,n论文 ΔIon,p论文
FBC(2 fin,5×18 nm)(基准)(基准) (基准)(基准)
SBC(2 sheet,18×5 nm)+7.7%+10% −9.1%−5%
SBC 宽(2 sheet,31×5 nm)+48.9%+73% +28.0%+47%

三种构型的亚阈摆幅落在 72–76 mV/dec。绝对电流不可直接 比较(取向迁移率比取文献典型值,而非论文的逐曲线 sub-band BTE 标定),但器件级趋势正确复现:SBC nMOS 因 (100) 面电子迁移率占优而 领先 FBC,SBC pMOS 因空穴迁移率劣势而落后,加宽 sheet 后 n/p 同时 提升。

4.3 数值交叉验证

除论文对比外,求解器内部做到位精确一致:不连通的器件区域按块对角 求解(已证明精确),复制沟道恰好承载单沟道电流的 N 倍(相对 误差 <10-6)。Windows 上采用与商用工具相同的 MKL PARDISO 直接求解器。

5. 局限

  • 无版图导入(GDSII/OASIS):本工具是器件级,处于工艺仿真的 上游。
  • 不与专有格式互操作:不读写 Sentaurus TDR/BND/CMD 文件。
  • 无应力迁移率模型:应变效应用手动迁移率标定 (mobility_scale_n/p)近似,而非物理应力求解器。
  • 无寄生 RC 提取或环振瞬态:电路级能力止于反相器 VTC (混合模式)。
  • 3D fin 用旋转 GAA 近似(四面环包),而非衬底上真实的 三栅。
  • 量子修正 3D 默认关闭(成本考虑);限域 Vt 偏移 重要时需显式开启。
  • 迁移率取向比取文献值,非逐曲线 BTE 标定 — 相对趋势 可靠,绝对电流为指示性。
